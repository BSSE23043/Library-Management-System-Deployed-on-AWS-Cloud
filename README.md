# Library-Management-System-Deployed-on-AWS-Cloud
This is my 5th semester group project for Software Development and Construction in Information Technology University, Lahore.

Tools & Technologies Used:
=> PERN Stack

=> Deployment on AWS:
  -> EC2 (Load Balanced & Auto Scaled Between 2 Subnet Groups)
  -> SNS
  -> Internet Gateway
  -> RDS (Postgres SQL Instance) 

=> Putty For SSH Access To EC2 Instances

=> VS Code For Code Editing
